# Keyboard Shortcuts Reference

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open Command Palette |
| `Cmd/Ctrl + Shift + /` or `?` | Show keyboard shortcuts |
| `Cmd/Ctrl + Z` | Undo last operation |
| `Cmd/Ctrl + Shift + Z` or `Cmd/Ctrl + Y` | Redo operation |
| `Cmd/Ctrl + S` | Save (if editing) |
| `Cmd/Ctrl + ,` | Open Settings |
| `Escape` | Close modal / Cancel action |

## Navigation

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate commits in graph |
| `←` / `→` | Navigate between parent/child commits |
| `Home` | Go to first commit |
| `End` | Go to working copy |
| `Enter` | Open selected commit details |
| `Tab` | Switch between panels |

## Commit Graph

| Shortcut | Action |
|----------|--------|
| `N` | Create new change |
| `E` | Edit selected change |
| `D` | Edit description |
| `A` | Abandon change |
| `S` | Squash into parent |
| `M` | Move/rebase change |
| `F` | Toggle bookmark filter |
| `G` | Go to commit (by ID or bookmark) |
| `/` | Focus search |
| `R` | Refresh graph |

## Working Copy

| Shortcut | Action |
|----------|--------|
| `C` | Commit working copy |
| `X` | Discard selected file changes |
| `I` | Add file to .gitignore |
| `O` | Open file in external editor |

## File Explorer

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate files |
| `Enter` or `Space` | Open file diff |
| `Delete` | Discard file changes |
| `F` | Filter files |
| `Shift + F` | Toggle filter panel |

## Diff Viewer

| Shortcut | Action |

| Shortcut | Action |
|----------|--------|

## Operations Panel

|----------|--------|

## Bookmarks

| Shortcut | Action |

## Search and Filter

## Command Palette

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate commands |
| `Enter` | Execute selected command |
| `Escape` | Close palette |
| `Backspace` | Clear search |

## Customization

You can customize keyboard shortcuts in Settings:

1. Open Settings (`Cmd/Ctrl + ,`)
2. Navigate to "Keyboard Shortcuts"
3. Click on a shortcut to edit
4. Press your desired key combination
5. Click "Save"

To reset to defaults, click "Reset to Defaults" at the bottom of the shortcuts panel.

## Platform-Specific Notes

### macOS

- Use `Cmd` instead of `Ctrl`
- `Option` key may be used for some shortcuts
- `Delete` key removes the character to the left

### Windows/Linux

- Use `Ctrl` for keyboard shortcuts
- `Alt` key may be used for some shortcuts
- `Delete` key removes the character to the right

### Accessibility

- All actions can be performed via keyboard
- Tab navigation follows logical order
- Focus indicators are visible
- Screen reader compatible
